# Responsive Card Slider

Um componente de carrossel/deslizador de cartões (Card Slider) moderno, limpo e totalmente responsivo desenvolvido com HTML5 e CSS3. O projeto simula uma seção de depoimentos ou exibição de perfis de equipe estruturada de forma flexível, garantindo uma navegação fluida em dispositivos móveis e resoluções compactas.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner do carrossel, dos cartões individuais de perfil e dos blocos de texto.
* CSS3: Estilização completa, criação de molduras circulares, gerenciamento da paleta de cores azul e definição de cantos arredondados.
* Flexbox / CSS Overflow: Organização linear dos cartões em um fluxo horizontal contínuo com suporte a rolagem fluida.

## Funcionalidades e Técnicas Aplicadas

* Layout Deslizante Otimizado para Mobile: Estruturação dos cards de forma paralela que permite a visualização parcial dos cartões adjacentes nas extremidades da tela, indicando intuitivamente a continuidade do conteúdo.
* Moldura de Avatar Dupla: Fotos de perfil estilizadas em formato perfeitamente circular com um contorno duplo elegante integrado à cor principal do tema.
* Barra de Paginação/Progresso: Inclusão de um indicador de paginação sutil na base inferior do contêiner para sinalizar o progresso ou a posição atual do carrossel.
* Estética Clean de Alto Contraste: Uso de cartões brancos com cantos suavizados e textos escuros de alta legibilidade, montados sobre um contêiner estrutural azul sólido.
* Experiência de Toque Nativa: Uso de propriedades CSS como `overflow-x: auto` e `scroll-snap-type` para proporcionar um efeito de deslizamento por toque (*swipe*) leve e responsivo.
